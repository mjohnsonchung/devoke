How it works

Three steps. Zero friction.

Features

Built for trust, not dependency.

  • Metric hiding

    Hide the score.

    Like counts, repost tallies, vote scores — these numbers are engineered to influence you. Devoke hides them by default. You decide what deserves your attention, not an algorithm.

  • Filter intensity

    Set your threshold.

    Light mode rewrites only the clearest manipulation. Deep mode catches mild loaded language and ambiguous framing. Balanced sits in between. Adjust any time from the popup.

  • Full transparency

    Always see the original.

    Every rewrite has a ◎ toggle. One tap shows you the original, word for word. Devoke is a translator — it never hides what was actually said.

Privacy

We don't want your data.

Devoke is a client-side extension. There are no Devoke accounts, no Devoke servers, and no analytics collecting your browsing habits.

  • ◎

    Your key, your calls

    Devoke uses your own Anthropic API key. Requests go directly from your browser to Anthropic — no proxy, no middleman, no Devoke server ever sees your data.

  • ◈

    Nothing is stored

    Devoke keeps an in-memory cache to avoid re-processing posts you've already seen. It's wiped when you close the tab. Nothing is written to disk or sent anywhere.

  • ◉

    Original is always intact

    We only rewrite what's displayed. The original HTML is preserved in memory for the toggle — we never modify or delete the content a platform published.
