Privacy Policy

Last updated: March 2026

Overview

Devoke is a browser extension that runs entirely on your device. We do not operate servers that receive, store, or process your browsing data. There are no Devoke accounts and no analytics.

What data Devoke handles

Text content from web pages

When you browse X, Reddit, news sites, or Google Search, Devoke reads the text of posts and headlines visible in your viewport. This text is sent to Anthropic's API using your own API key to generate a rewritten version. Devoke does not store this text anywhere — it passes through in-memory only and is discarded when you close the tab.

Your Anthropic API key

Your API key is stored using chrome.storage.sync, which is managed by Chrome and may sync across your Chrome profile. The key is only ever sent to Anthropic's API endpoints — never to any Devoke server, because no Devoke server exists. Do not share your API key with anyone.

Extension settings

Your settings (enabled state, filter intensity, metric hiding preference) are stored in chrome.storage.sync. These preferences are local to your Chrome profile and are never transmitted anywhere.

Third-party services

The only third-party service Devoke communicates with is the Anthropic API (Anthropic Privacy Policy). Requests are made directly from your browser to Anthropic using your API key. Anthropic's data handling is governed by their own privacy policy.

Devoke does not use Google Analytics, Sentry, Mixpanel, or any other analytics or error-tracking service.

Browser permissions

Devoke requests the following Chrome permissions:

  • Host permissions for X, Reddit, supported news sites, and Google Search — required to inject the content script that reads and rewrites page text.
  • storage — to save your API key and settings via chrome.storage.sync.

Devoke does not request history, cookies, tabs, or any other broad permission.

Children's privacy

Devoke is not directed at children under 13 and does not knowingly collect information from children.

Changes to this policy

If this policy changes in a material way, we will update the "Last updated" date at the top of this page. Continued use of the extension constitutes acceptance of the updated policy.
